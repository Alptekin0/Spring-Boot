# Spring Boot Data JPA

## Proje Açıklaması
Bu proje, **Spring Boot** ve **Spring Data JPA** kullanılarak öğrenme amaçlı geliştirilmiştir.  
Temel hedef; veritabanı işlemlerini JPA aracılığıyla yönetmeyi, katmanlı mimariyi kavramayı ve   
RESTful API mantığını uygulamalı olarak öğrenmektir. Proje boyunca CRUD operasyonları, repository yapısı ve entity–DTO ayrımı gibi temel backend kavramları  
ele alınmıştır.
