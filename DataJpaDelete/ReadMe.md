## 🗑️ Spring Boot Data JPA Delete API

Bu proje, Spring Boot ve Spring Data JPA kullanarak veritabanındaki **öğrenci kaydını silme (DELETE)** işlemini gerçekleştiren basit bir REST API uygulamasıdır.

### 🚀 Özellikler
- Öğrenci kaydını **ID ile silme**
- RESTful API yapısı
- Katmanlı mimari (Controller - Service - Repository)
- PostgreSQL veritabanı entegrasyonu

### 🛠️ Kullanılan Teknolojiler
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok

### 🔎 Açıklama
id: Silinmek istenen öğrencinin ID değeri  
Eğer kayıt varsa silinir ve true döner  
Kayıt yoksa false döner
