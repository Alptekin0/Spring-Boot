# 🚀 Spring Boot Fundamentals Project

Bu proje, Spring Boot’un temel yapısını incelemek ve öğrenmek amacıyla geliştirilmiştir.  
Uygulama; katmanlı mimari, REST API yapısı ve temel Spring Boot konfigürasyonlarını içermektedir.  
  
# 📌 Proje Amacı  
  
- Spring Boot proje yapısını anlamak  
- Controller, Service ve Repository katmanlarını incelemek  
- RESTful API geliştirme mantığını kavramak  
- application.properties/yml konfigürasyonlarını öğrenmek  
  
# 🛠️ Kullanılan Teknolojiler  
  
- ☕ Java  
- 🌱 Spring Boot  
- 📦 Maven  
- 🌐 Spring Web  
- 🗄️ Spring Data JPA
