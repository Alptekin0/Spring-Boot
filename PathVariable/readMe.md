# Spring Boot @PathVariable Pratik Projesi

Bu proje, Spring Boot framework'ünde `@PathVariable` anotasyonunun kullanımını öğrenmek ve pratik yapmak amacıyla oluşturulmuş basit bir uygulamadır.

## 📌 Proje Hakkında

Uygulama içerisinde temel bir Öğrenci (Student) REST API yapısı (Entity, Repository, Service ve Controller) kurgulanmıştır. Temel odak noktası, kullanıcıdan gelen isteklerde URL içerisine yerleştirilen değişkenlerin (path variables) Controller katmanında nasıl yakalandığını ve işlendiğini anlamaktır.

Bu proje tamamen **eğitim, öğrenme ve pratik yapma** amacıyla geliştirilmiştir. 

## 🛠️ Kullanılan Teknolojiler

* Java
* Spring Boot
* Spring Web (RESTful API mimarisi)
* Spring Data JPA
